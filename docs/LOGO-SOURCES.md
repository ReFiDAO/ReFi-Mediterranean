## Logo Requirements

- **Format**: PNG with transparent background preferred, or SVG
- **Size**: Minimum 200px wide for web use
- **Usage**: Ensure compliance with each organization's brand guidelines
- **Storage**: Save to `assets/partner-logos/` directory

## Notes

- Always respect brand guidelines when using partner logos
- Some logos may require permission for use
- Ensure logos are current/up-to-date versions
- Consider creating a "Partners" section on the website with proper attribution
